# Healthcare data analysis on OCD Patients

## Project overview
Analyze healthcare data of OCD patients using SQL to identify pattern and trend

## Tools used
MYSQL
EXCEL

## Questions answered were on:
1. Which gender visited the facility often and what was their respective average obs score?
2. The percentage of patients in both gender that visited the facility?
3. Total number of visitors on monthly basis?
4. The most common obs type and its corresponding score?
5. The most common comp type and its corresponding comp scores

   ## Key Findings
   1. Female gender had the highest patient count
   2. Both genders had the same average obs score
   3. The higest month of patient count was July 2014
   4. Washing had the highest comp score from its patients count
   5. Religious recorded the highest obsession type
